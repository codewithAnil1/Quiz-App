### ğŸ“‹ Prerequisites

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-github-username/quiz-master.git
   cd quiz-master
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```
   or
   ```bash
   yarn install
   ```

3. **Start the Development Server**:
   ```bash
   npm run dev
   ```
